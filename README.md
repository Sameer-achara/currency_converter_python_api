## 🌍 Supported Currencies

This project supports **all major international currencies** available via ExchangeRate-API.  
Some commonly used currency codes include:

- USD → United States Dollar  
- INR → Indian Rupee  
- EUR → Euro  
- GBP → British Pound  
- AUD → Australian Dollar  
- CAD → Canadian Dollar  
- JPY → Japanese Yen  
- CNY → Chinese Yuan  
- CHF → Swiss Franc  
- NZD → New Zealand Dollar  

👉 You can use **any valid ISO 4217 currency code** (e.g., `USD`, `INR`, `EUR`, etc.) for conversion.
